# ISO 8583

ISO 8583 is a NodeJs library for dealing with ISO message from and to custumer merchant.

## Installation

Clone de repository


## Usage

```html
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
